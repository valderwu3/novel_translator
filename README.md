## 注意事项：

### 章节分割符：
可以写多个，以其中任何一个开头的行都会被认为是章节起始行，两个分割行直接的内容被识别为一章。

### API KEY:
目前只支持deepl api，API key 有两种获取方式：
第一种是自己到DEEPL 官网申请，免费API额度，一个月50万字符，需要非大陆信用卡，香港，台湾信用卡也是可以的。
第二种是直接某宝购买账号，搜索“deepl api”就能找到，大概8块3+个月（具体+多少看老板良心）。
